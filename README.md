# ProgParaJogosPong
Jogo projeto para cadeira de Programação para Jogos

# PONG

## Identidade do Jogo

Pong é um jogo simples de tênis, onde rebater a bola com as raquetes (plataformas) e marcar pontos definirão o jogador(a) vencedor(a).

## Descrição da Mecânica

Um jogo 2.5 (2D de perspectiva com sprites 3D), envolvendo o rebote de uma bola e duas plataformas.

## Características

Um jogo de rebote de bola, evolvendo duas plataformas laterais e uma bola.

## Arte

Jogo 2D com sprites 3D.

## Música/Trilha Sonora

Ainda não foi visto a necessidade de uma trilha sonora para o jogo.

## Interface/Controles

Haverá um botão de pause, um placar para definir o vencedor.

## Dificuldade

A dificuldade será definida pela habilidade dos jogadores.

## Personagem

Duas raquetes e uma bola.

## Cronograma

Existe a possiblidade de ter níveis (que serão definidos por obstáculos aleatórios na tela)

## Definições Gerais

Gênero: Plataforma 2D
